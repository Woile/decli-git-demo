decli-git-demo
==============


This is the code used in the tutorial explaining `decli <https://github.com/Woile/decli>`_ in my `personal blog <http://woile.github.io/>`_.

Installation
------------

.. warning::

    Everything here is python3!

First clone it.

Poetry
~~~~~~

::

    poetry install

Pip
~~~

::

    python -m venv venv
    source venv/bin/activate
    pip install -U decli

Usage
-----

::

    python -m git --help
